# Strewt

**Strewt** - or the String Reading & Writing library - provides reader and writer utilities for interpreting and writing text. To keep expensive string copying and combining to a minimum, both reading and writing use GM buffers under the hood. While they may not be as efficient as a well-written dedicated implementation - let alone an extension - they should perform reasonably well for a general-purpose solution and work on any GM supported platform.
